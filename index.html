Sobre Mim

Olá recrutador, eu me chamo Carlos Alberto e sou um programador generalista além de empreendedor. Comecei a me interessar por programação ainda quando era mais novo tendo contato com VisualBasic sem nem saber que estava programando algo na época e em seguida aprendi Python por ter um grande interesse no mercado de segurança de informações na Internet e era a mais recomendada pelos meus meios de estudo porém nunca realmente levei isso a sério e cheguei até a parar de programar. Recentemente eu despertei interesse por desenvolvimento web/mobile e com todo grande conhecimento em Python eu aprendi Django Framework e Django Rest Framework, porém me senti limitado por não ser capaz de utilizar a mesma linguagem para construir um front-end (com um React ou React Native por exemplo) e então estou estudando continuamente a stack de JavaScript e futuramente conseguir estar apto para resolucionar problemas como um desenvolver FullStack utilizando Node e React.

Atualmente estou procurando uma vaga como desenvolvedor em JavaScript ou Python para conseguir unir o meu trabalho com meus estudos e iniciar minha faculdade. Desenvolver códigos para mim não é trabalho, é uma diversão séria de adulto, consigo passar horas estudando ou trabalhando que não percebo a hora passar tão rápida.

Experiências

Eu atualmente já trabalho como desenvolvedor freelancer na plataforma da 99freelas e isso tem me dado bastante experiência para lidar com clientes e formas de deixar os códigos mais limpos e elegíveis para features conforme a demanda do cliente além de ter desenvolvido bastante minha forma de lidar com clientes, vendas e outras estratégias relaciondas.

Fiz um projeto para Magazine Luiza que atualmente é usado. O projeto é um site que prospecta possíveis clientes raspados GoogleMaps e envia para o banco de dados que é disponibilizado através de uma interface para o usuário autenticado. Para autenticação desse projeto utilizei a mesma API que a Magazine Luiza utiliza em uma de suas ferramentas internas para os colaboradores que requer autenticação deles, para a raspagem dos dados utilizei Python com Selenium e por fim a construção do site foi utilizado Django Framework mas estou atualmente trabalhando para deixar tudo isso com mais performance e migrar o back-end para o NodeJS.
